# Website - Escola Burille

Projeto de desenvolvimento de site institucional para a Burille | Escola de Artes Marciais.

## Descrição
Desenvolvimento de um website com objetivo de apresentar a escola, modalidades
de artes marciais, informações de contato e loja.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- React

## Funcionalidades
- Página inicial
- Informações sobre modalidades
- Página de contato
- Loja
- Layout responsivo

## Observação
O código-fonte deste projeto é privado por se tratar de um projeto desenvolvido
para uma instituição real.

## Demonstração
Link do site: https://escola-burille.vercel.app/

## Status
Projeto concluído.
